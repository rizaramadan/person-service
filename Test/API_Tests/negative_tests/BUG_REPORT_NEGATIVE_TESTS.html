üìã Description

üß™ Failing Test Cases

  3. GET /api/key-value/{2000-char-key} ‚Üí 500 Error

‚ùå Impact

  • Security Risk: Bisa digunakan untuk DoS attack
  • Server crash saat memproses input yang sangat besar

  • Database bisa overflow dengan data besar
  • Response time sangat lambat untuk request berikutnya

üîß Recommended Fix

// Add length validation const ( MAX_KEY_LENGTH = 1000 // 1KB max MAX_VALUE_LENGTH = 100000 // 100KB max ) // Validate before processing if len(request.Key) > MAX_KEY_LENGTH { return c.JSON(400, map[string]string{ "error": "Key too long", "message": fmt.Sprintf("Key length exceeds maximum of %d characters", MAX_KEY_LENGTH), }) } if len(request.Value) > MAX_VALUE_LENGTH { return c.JSON(413, map[string]string{ "error": "Payload too large", "message": fmt.Sprintf("Value length exceeds maximum of %d characters", MAX_VALUE_LENGTH), }) }
